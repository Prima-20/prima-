表格 #

功能 #

  1. 可配置型表格，通过配置对象的形式自动生成表格
  2. 可编辑单元格，每次只能编辑一个单元格
  3. 可编辑行，每次只能编辑一行
  4. 具备element-plus原有表格的所有功能

用法示例 #

vue
<template>
  <p-table
    stripe
    ref="mtableRef"
    :options="options"
    :data="tableData"
    isEditRow
    isPagination
    :total="total"
    :currentPage="current"
    :pageSize="pageSize"
    :pageSizes="[10, 15, 20]"
    paginationAlign="center"
    @currentChange="currentChange"
    @sizeChange="sizeChange"
  >
    <template #date="{ scope }">
      <el-icon-timer></el-icon-timer>
      {{ scope.row.date }}
    </template>
    <template #name="{ scope }">
      <!-- 从elm复制来的，鼠标悬停时有气泡框提示 -->
      <el-popover effect="light" trigger="hover" placement="top">
        <template #default>
          <p>姓名: {{ scope.row.name }}</p>
          <p>住址: {{ scope.row.address }}</p>
        </template>
        <template #reference>
          <div class="name-wrapper">
            <el-tag size="small">{{ scope.row.name }}</el-tag>
          </div>
        </template>
      </el-popover>
    </template>
    <template #action="{ scope }">
      <el-button type="primary" size="small" @click="edit(scope)"
        >编辑</el-button
      >
      <el-button type="danger" size="small">删除</el-button>
    </template>
    <template #editCell="{ scope }">
      <div class="edit">
        <el-button type="primary" size="small" @click="confirm(scope)"
          >确认</el-button
        >
        <el-button size="small" @click="cancel(scope)">取消</el-button>
      </div>
    </template>
    <template #editRow="{ scope }">
      <el-button type="primary" size="small" @click="confirmEditRow(scope)"
        >确认</el-button
      >
      <el-button size="small" @click="cancelEditRow(scope)">取消</el-button>
    </template>
  </p-table>
</template>

<script setup lang="ts">
import { onMounted, ref, watch } from "vue";
import { TableOptions } from "prima-elm-cps/table";
import axios from "axios";

interface TableData {
  date: string;
  name: string;
  address: string;
}

let tableData = ref<TableData[]>([]);

let current = ref<number>(1);
let pageSize = ref<number>(10);
let total = ref<number>(0);
//获取分页的数据
const getData = () => {
  axios
    .post("/api/list", { current: current.value, pageSize: pageSize.value })
    .then((res) => {
      if (res.data.code === "200") {
        tableData.value = res.data.data.rows;
        total.value = res.data.data.total;
      }
    });
};
onMounted(() => {
  getData();
});

//表格配置
let options: TableOptions[] = [
  {
    prop: "date",
    label: "日期",
    // width: '180',
    align: "center",
    slot: "date",
    editable: true, //单元格是否可编辑
  },
  {
    prop: "name",
    label: "姓名",
    // width: '180',
    align: "center",
    slot: "name",
  },
  {
    prop: "address",
    label: "地址",
    align: "center",
    editable: true,
  },
  {
    label: "操作",
    action: true, //是否是操作项
    align: "center",
  },
];

const confirm = (scope: any) => {
  console.log("编辑行确认", scope);
};
const cancel = (scope: any) => {
  console.log("编辑行取消", scope);
};

//获取子组件节点
let mtableRef = ref();
//点击操作项的编辑按钮显示编辑行
const edit = (scope: any) => {
  mtableRef.value.showEditRow(scope.row.date);
};
//编辑行确认
const confirmEditRow = (scope: any) => {
  scope.row.currentEditRow = false;
  console.log(scope);
};
//编辑行取消
const cancelEditRow = (scope: any) => {
  scope.row.currentEditRow = false;
};
//当前页数改变
const currentChange = (val: number) => {
  current.value = val;
  getData();
};
//每页显示条数改变
const sizeChange = (val: number) => {
  pageSize.value = val;
  getData();
};
</script>

<style scoped lang="scss">
.edit {
  display: flex;
  margin-left: 12px;
}
</style>

属性 #

属性名说明类型默认值
options表格配置TableOptions[]—
data表格数据any[]—
editIcon编辑单元格显示的图标string"Edit"
isEditRow是否可以编辑行booleanfalse
isPagination是否需要分页器booleanfalse
total总页数number—
pageSize每页显示多少条数据number10
pageSizes可供选择的pageSize组成的数组Number[][5, 10, 15]
currentPage当前页码number1
paginationAlign分页器排列布局"left" |"center"|"right""center"

事件 #

事件名说明回调参数
confirm编辑单元格时，点击默认的"√"图标触发（scope: any）
cancel编辑单元格时，点击默认的"×"图标触发（scope: any）
currentChange当前页码发生变化（newPage：number）
sizeChange每页显示数据条数变化（newPageSize：number）

插槽 #

除以下插槽外，还有一些动态的插槽，插槽名根据用户传进来的slot属性决定。

插槽名描述说明
editCell编辑单元格时，自定义图标或按钮的位置
editRow编辑行时，切换自定义图标或按钮的位置
action按钮操作区的位置

对外暴露的方法 #

名称详情类型参数
showEditRow显示编辑行function当前行的date属性
Previous page弹出框表单
